## 部署 Cloudflare Worker

逐步引導：
1. 前往 dash.cloudflare.com → Workers & Pages → Create application
2. 選 Start with Hello World
3. 先 Deploy
4. 點 Edit code → 貼上 worker.js 內容 → Deploy
5. Settings → Variables and Secrets → 新增 Secret：`CLAUDE_API_KEY`

---

## 取得 Claude API Key

- 前往 console.anthropic.com → API Keys → Create Key
- 選 Individual 帳號
- 填入名稱（accessibility-tool）→ Add → 複製 Key

---

## 設定 WORKER_URL

將 Cloudflare Worker 網址填入 index.html：
```js
const WORKER_URL = 'https://weathered-boat-76be.love520297.workers.dev';
```

## 功能改進

### 1. 頁面現有標題層級設定

**User：** 如果頁面固定架構已有 h1 + h2，內容不應該從 p 變成 h1。
